#### 简介

首先感谢追书神器, 这里的接口都是用的追书神器的(除了用户) 由于不是收费接口所以这里就不对接口数据进行存储了,都是直接请求追书的接口

如果是收费接口建议还是要做下缓存的(接口没有做任何缓存)


#### 关于axios

这里仅仅是axios的初级应用,并没有新建实例,因为我发现追书的接口都是直接暴露的,我这里并不需要追加header之类的东西,所以还是直接引入使用方便
