# 追书神器

该项目是基于追书神器的接口二次封装，线上演示地址： https://novel.juhe.im。 文档： https://xiadd.github.io/zhuishushenqi/

如有需要建议自行打包部署

## 依赖

koa，react，nextjs等等，不一一列出

## 开发

```
yarn dev
```

## 打包部署

```
// 建议本地打包
yarn build 
yarn start
```
